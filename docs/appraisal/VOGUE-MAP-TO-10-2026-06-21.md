# Vogue — the map to 10/10 (£200m flagship) — 2026-06-21

> A design-house roadmap from the current **7.7/10** craft to a flawless 10.
> Synthetic critique (a "world-class design house" persona), rendered against the
> real v3.127.3 build. Hard constraints honoured throughout: clinical-alert
> salience (red/amber) is sacred and never dimmed for aesthetics; the brand cyan
> stays out of the clinical chrome; colour stays spent on clinical status.

## 1. What "10" means for this product

A 10 is not a beautiful app; it is **an instrument that disappears**. For a 400px
clinical side-panel: a GP glances between consultations and the single most
important thing is already the loudest thing on screen, zero parsing cost. Total
**salience discipline** (red/amber only for clinical urgency), one coherent
**component grammar** (a chip is a chip everywhere; a number is weighted the same
everywhere), chrome so quiet it reads as confidence not decoration. The reference
is a Leica viewfinder: every line earns its place, the frame is mostly empty, the
one number you need is etched not boxed. Nothing is there "to fill the card";
nothing shouts that doesn't bleed.

## 3. The map — phased 7.7 → 10

### Phase 1 — "Stop shouting" (7.7 → 8.3) · pure subtraction, highest leverage
1. **Kill all-caps mono on action buttons** (Choose tabs / Expand / Dismiss /
   Open full report / Run the pre-clinic sweep / Copy figures / CSV / Today / 7
   days / 30 days) → sentence-case sans. Mono stays on data/nav/status. **[S]**
2. **Cull the accent palette to two families** — clinical = red/amber/(green for
   clear); structural/numeric = one neutral ink + brand cyan only as a quiet
   link colour, never a status. Collapse the five-hue triage-count numerals to
   one neutral weight-based emphasis. **[M]**
3. **Demote the "Open →" affordance** — hover/focus reveal or a single chevron,
   not standing text on all six Today cards. **[S]**
4. **Fix the truncated "Practice code set · 3 …" banner.** **[S]**
5. **Unify number treatment** — one numeral scale system (hero / sub / inline)
   across cards. **[S]**

### Phase 2 — "One grammar" (8.3 → 9.0) · component convergence
1. **Converge the three pill dialects into one capsule spec** with status
   variants (triage capsule, WR red-border chip, slot outline chip → one
   shape/radius/padding/type, differentiated only by a status token). **[L]**
2. **Recompose Today from six equal cards into a composition** (the marquee
   move): promote Waiting Room + Max wait to a single clinical hero; demote
   Demand / Slots / Triage Load to a compact stat row (label + numeral, no card
   chrome); keep Morning Sweep as the one CTA. One hero, one stat band, one
   action — not six equal tiles. **[XL]**
3. **Strip card chrome to surfaces** — replace border + header-rule + "Open" with
   quiet elevation or a single hairline divider. **[L]**
4. **Redraw the Condor mini bar-charts to the owned-meter language** (same tick
   rhythm, restrained 2-hue fill, same baseline). **[L]**

### Phase 3 — "Resolve" (9.0 → 9.7) · system tuning
1. **One baseline grid** for vertical rhythm across Today / Condor / Slots. **[M]**
2. **Dark-mode elevation/colour tuning** to mirror the new light hierarchy. **[M]**
3. **Slots "By clinician" rows** to a fixed am/pm/total micro-grid so the eye
   scans a column, not a sentence. **[M]**

## 4. The last mile (9 → 10) — the micro-craft teams skip

- **Optical (not metric) alignment** of the reticle mark, hero numerals, chip
  baselines; reconcile mono-numeral baselines with adjacent sans labels.
- **Tabular-lining numerals** everywhere a column of numbers exists (Slots
  clinician rows, WR wait times, Condor figures) so digits don't jitter.
- **One owned transition curve** — same easing/duration token for every
  reveal/hover/expand; verify the resting state is the truth.
- **Focus states as designed objects** — a 2px brand-cyan ring with per-component
  radius, consistent across buttons/chips/tabs/date input.
- **Hairline discipline** — one border colour, one weight, sub-pixel aware.
- **Top WR strip shares the body's exact pill grammar and red/amber tokens** (they
  currently differ subtly).
- **Empty-state typography parity** (measure, leading, grey value).
- **Pixel-snapping** of the Condor meter ticks and chevrons so nothing blurs at 1x.

## 5. Definition of done for 10 — reviewer's checklist

- [ ] Zero all-caps mono outside data/nav/status — no shouting buttons.
- [ ] One pill spec, one button spec, status-variant only — a chip looks identical
      on Today, Condor, Slots and the top strip.
- [ ] Red/amber appear ONLY for clinical urgency; cyan never a status; ≤ 3 accent
      hues on Today.
- [ ] Today reads as one composition: one clinical hero, a quiet stat band, one
      CTA — not six equal cards.
- [ ] No card carries border + header + standing "Open" at once; chrome is
      elevation/hairline.
- [ ] Every chart shares the Condor meter's visual language; no stock defaults.
- [ ] All number columns tabular-aligned; hero numerals optically centred.
- [ ] One spacing unit, one border weight, one transition curve, one focus-ring
      spec across all surfaces.
- [ ] Dark mode mirrors the light hierarchy; clinical red holds full salience.
- [ ] Glance test: an unprompted GP names the most clinically urgent item within
      one second, no competing element of equal weight.

**Headline:** the points left are in **composition, chrome subtraction and colour
discipline**, not type or motion. Phase 1 is pure subtraction and buys the most
per hour; Phase 2's Today recomposition + pill convergence is the structural
heavy-lift that separates an arrangement from a £200m instrument.
